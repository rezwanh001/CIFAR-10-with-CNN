# CIFAR-10-with-CNN

**CIFAR10 small image classification.**


Dataset includes:

1.   50,000 training examples
2.   10,000 testing examples
3.   10 classes
4.    (3, 32, 32) == (depth, rows, cols) images


The ten class labels include:

1.  Airplane
2.  Automobile
3.  Bird
4.  Cat
5.  Deer
6.  Dog
7.  Frog
8.  Horse
9.  Ship
10. Truck
